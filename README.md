# MemoryBench Research Workspace

> Track 1 (MemoryAgent) — Global AI Hackathon with Qwen Cloud

A four-store memory layer for LLM agents (episodic / semantic / preference /
working) with contradiction resolution, importance-and-recency decay, and
knapsack-based budgeted retrieval — plus **MemoryBench**, a benchmark harness
that proves it beats no-memory, full-history, and naive top-k RAG baselines
on recall, staleness, tokens/turn, cost, and latency.

_This README is filled in progressively as the project is built; see
[`docs/submission.md`](docs/submission.md) for the full submission writeup._
